# Chatbot Project

A simple chatbot application built with React.

## Features
- User can send messages to the chatbot.
- Chatbot responds with predefined answers.
- Messages are saved in `localStorage` so they persist after page refresh.
- Clear button resets the chat to the welcome message.

## How to Run
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

## Start the application
   ```bash
   npm run dev
   ```

## Technology Stack
- React
- Day.js (for time formatting)
- Vite (development server)
